📡 Modo offline (iPhone)

Com internet, abra esta página uma vez pelo menos para o sistema guardar uma cópia. Depois, sem Wi‑Fi/dados, o app pode abrir do cache.

No iPhone, o Service Worker só funciona se você abrir o arquivo por HTTPS (site, nuvem com link https, ou computador na rede com https). Abrir o .html direto do Arquivos (file://) não ativa o cache automático — seus dados (XP, tarefas, foto) continuam salvos no aparelho via localStorage em qualquer caso.
